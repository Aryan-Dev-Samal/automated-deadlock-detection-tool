\# Automated Deadlock Detection Tool



\## Project Overview

This tool is designed to simulate an Operating System environment and automatically detect deadlock situations using a matrix-based approach (similar to the Banker's Algorithm).



\## Team Members

\* Aryan Dev Samal

\## Modules

1\. \*\*Input Management:\*\* Captures System State (Available, Max, Allocation).

2\. \*\*Detection Engine:\*\* Analyzes the Wait-For Graph/Safety Sequence to find cycles.

3\. \*\*Recovery System:\*\* Suggests victim preemption based on maximum resource allocation.



\## How to Run

Compile `main.cpp` using any standard C++ compiler (e.g., g++) and run the executable. Follow the on-screen menu.
